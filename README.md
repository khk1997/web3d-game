# web3d-game

A stylized 3D driving prototype built with `Three.js`, `Rapier`, and `Vite`.

## Live Demo

GitHub Pages:

[https://khk1997.github.io/web3d-game/](https://khk1997.github.io/web3d-game/)

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Controls

- `W` / `Arrow Up`: move forward
- `S` / `Arrow Down`: reverse
- `A` / `D` or `Arrow Left` / `Arrow Right`: steer
- `R`: reset vehicle
- mouse drag: orbit camera

## Deployment

This repo is configured to deploy automatically to GitHub Pages on every push to `main`.
